# 🎬 StreamX Pro

A Netflix-inspired movie streaming UI built using HTML, CSS, and JavaScript.  
This project uses TMDB API to fetch real-time trending movies and display trailers.

---

## 🔥 Features

- 🎥 Trending Movies (Live from TMDB API)
- 🔍 Search Movies
- ▶️ Watch Trailer (YouTube integration)
- 🎨 Netflix-style UI design
- ✨ Hover animation (movie title + rating)
- 📱 Responsive layout

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- TMDB API
